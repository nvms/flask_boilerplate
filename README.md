# Requirements
- python 3.5
- flask (tested on 0.11.1)
- nodejs (tested on 4.6.1)
- gulp (tested on 3.9.1)

# Quickstart
- Clone the repository
- Install dependencies: ```npm install```
- ```python run.py```
- http://127.0.0.1:5000

# Development
- Compile CSS and watch for changes ```gulp sass:watch```
